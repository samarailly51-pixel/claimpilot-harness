How the harness works.

The framework is adapter-first and case-pack-first, so teams can plug in real agents without changing the evaluation surface.

1

Load case pack

Structured JSON cases define claimant context, policy constraints, evidence, traps, and expected safe behavior.

2

Connect agent

Use built-in baselines, command adapters, HTTP services, or OpenAI-compatible chat completions endpoints.

3

Score decisions

Deterministic scoring checks business action, evidence grounding, missing documents, and safety failures.

4

Replay failures

HTML reports show the evidence timeline, decision JSON, scoring breakdown, and exact weak points.
