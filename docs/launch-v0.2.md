# ClaimPilot Harness v0.2 Launch Kit

## One-line launch

ClaimPilot is an open-source crash-test lab for insurance claim Agents, now with a traceable Model Arena, human review workflow, ten adversarial cases, and an auto bodily injury domain Skill.

## Short social post

Insurance Agent demos are easy. Reliable claim decisions under missing documents, conflicting evidence, privacy traps, and prompt injection are not.

ClaimPilot Harness v0.2 now includes a Model Arena with dataset fingerprints and replayable results, a local-first Human Review Workbench, ten adversarial claim cases, and a reusable bodily injury claims Skill.

Demo: https://samarailly51-pixel.github.io/claimpilot-harness/

GitHub: https://github.com/samarailly51-pixel/claimpilot-harness

## Demo sequence

1. Select `auto-bodily-injury-001`.
2. Run the cautious baseline and open its replay.
3. Compare rule baselines in Model Arena.
4. Save a different human decision in Human Review Workbench.
5. Export the review JSON.
